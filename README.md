# What the hell is this

I made a convenient mod for minecraft; you can pick items in rectangle area and move to another inventory.<br>
Works with NeoForge 1.21.1 for now.

Compatible with Creative Tab, AE2 Terminal, Filter inventory like List filter of Create, and so on.

By default, drag-and-drop with C selects items in an area, and pressing C on another inventory moves the items.
